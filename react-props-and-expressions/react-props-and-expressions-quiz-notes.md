# react-props-and-expressions-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What are props in React?
  Information that you pass to a JSX tag.
- How do you declare and access props in a component?
  Within the functions parameter parenthesis, add curly braces and place the prop there.
- How do you pass props to a component?
  Props are usually passed from a parent component to a child component by importing it to the app.
- How do you write JavaScript expressions in JSX?
  Within curly braces.
